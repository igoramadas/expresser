App

This is the "core" of an Expresser based application. The App wraps the Express server and its middlewares / routes, along with a few extra helpers.

Errors

This is a helper class to make it easier to keep error management and exception handling consistent across the app.

Events

Central event manager to dispatch events to other Expresser modules. Acts as a wrapper to Node's builtin EventEmitter class.

Logger

Handles server logging using the console, local files, Logentries, Loggly and other transports available as plugins. Multiple transports can be enabled at the same time.

Settings

All main settings for an Expresser apps are set and described on the settings.default.json file. Do not edit it!!! To change settings please create a settings.json on the root of your application and put your own values there.
