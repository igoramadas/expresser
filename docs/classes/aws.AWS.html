AWS

Module to integrate your app with Amazon Web Services using the official AWS SDK module. As of now it implements helpers for DynamoDB, S3 and SNS, but you can use any of the AWS SDK features by accessing the sdk of this module directly.

Please see https://aws.amazon.com/sdk-for-node-js/ for the AWS SDK docs.

    logger.debug "AWS.init"

    # Init the implemented AWS modules.
    @dynamodb.init this
    @s3.init this
    @sns.init this

    events.emit "AWS.on.init"
    delete @init
