MAILER TEMPLATES

Helper class to load and parse email templates.

Instance Method Details

# (void) init(parent)

Init the Templates class.

# (String) get(name)

Load and return the specified template. Get from the cache or from the disk if it wasn't loaded yet. Templates are stored inside the /emailtemplates folder by default and should have a .html extension. The base template, which is always loaded first, should be called base.html by default. The contents will be inserted on the {contents} tag.

Parameters:

  • name ( String ) — The template name, without .html.

Returns:

  • ( String ) — The template HTML.

# (String) parse(template, keywords)

Parse the specified template to replace keywords. The keywords is a set of key-values to be replaced. For example if keywords is {id: 1, friendlyUrl: "abc"} then the tags {id} and {friendlyUrl} will be replaced with the values 1 and abc.

Parameters:

  • template ( String ) — The template (its value, not its name!) to be parsed.
  • keywords ( Object ) — Object with keys to be replaced with its values.

Returns:

  • ( String ) — The parsed template, keywords replaced with values.

# (void) clearCache()

Force clear the templates cache.
