# (void) newInstance()

# (String) getIP(firstOnly)

Returns a list of valid server IP addresses. If firstOnly is true it will return only the very first IP address found.

Parameters:

  • firstOnly ( Boolean ) — Optional, default is false which returns an array with all valid IPs, true returns a String will first valid IP.

Returns:

  • ( String ) — The server IPv4 address, or null.

# (Object) getInfo()

Return an object with general information about the server.

Returns:

  • ( Object ) — Results with process pid, platform, memory, uptime and IP.

# (Object) getCpuLoad()

Get current CPU load (used mainly by getServerInfo).

Returns:

  • ( Object ) — CPU load with idle and total ticks.
