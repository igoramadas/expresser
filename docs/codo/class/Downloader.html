# (Object) download(remoteUrl, saveTo, options, callback)

Download an external file and save it to the specified location. The callback has the signature (error, data). Returns the downloader object which is added to the queue, which has the download properties and a stop helper to force stopping it. Returns false on error or duplicate. Tip: if you want to get the downloaded data without having to read the target file you can get the downloaded contents via the options.downloadedData.

Parameters:

  • remoteUrl ( String ) — The URL of the remote file to be downloaded.
  • saveTo ( String ) — The full local path and destination filename.
  • options ( Object ) — Optional, object with request options, for example auth.
  • callback ( Method ) — Optional, a function (err, result) to be called when download has finished.

Returns:

  • ( Object ) — Returns the download job having timestamp, remoteUrl, saveTo, options, callback and stop helper.

# (Boolean) stop(remoteUrl, saveTo)

Force stop a current download.

Parameters:

  • remoteUrl ( String ) — The URL of the download to stop.
  • saveTo ( String ) — The full local path of the download to stop.

Returns:

  • ( Boolean ) — Returns true if a match was found, or false if no matching download to stop.
