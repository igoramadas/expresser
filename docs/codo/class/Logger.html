Class: Logger

Defined in: lib/logger.coffee

Overview

EXPRESSER LOGGER

Handles server logging using local files, Logentries, Loggly and other transports available as plugins. Multiple services can be enabled at the same time.

See also:

  • settings.logger

Property Summary

(Object) drivers
Holds a list of available logging drivers.
(Object) transports
List of registered transports.
(Method) onLogSuccess
Custom (result) method to call when logs are sent to logging server or flushed to disk.
(Method) onLogError
Custom (err) method to call when errors are triggered by the logging transport.
