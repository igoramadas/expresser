Overview

AWS S3

Handles communication with AWS S3.

Instance Method Summary

  • # (void) init(parent) Init the S3 module.
  • # (void) download(bucket, key, destination, callback) Download a file from S3 and optionally save to the specified destination.
  • # (void) upload(bucket, key, body, options, callback) Upload a file to S3.
  • # (void) delete(bucket, keys, callback) Delete object(s) from S3.

Instance Method Details

# (void) init(parent)

Init the S3 module.

# (void) download(bucket, key, destination, callback)

Download a file from S3 and optionally save to the specified destination.

Parameters:

  • bucket ( String ) — Name of the S3 bucket to download from.
  • key ( String ) — Key of the target S3 bucket resource (usually a filename).
  • destination ( String ) — Optional, full path to the destination where file should be saved.
  • callback ( Method ) — Callback (err, body), with body being the contents as String.

# (void) upload(bucket, key, body, options, callback)

Upload a file to S3.

Parameters:

  • bucket ( String ) — Name of the S3 bucket to upload to.
  • key ( String ) — Key of the target S3 bucket resource (usually a filename).
  • body ( Object ) — Contents of the file to be uploaded.
  • options ( Object ) — Upload options.
  • callback ( Method ) — Callback (err, result).

Options Hash: (options):

  • acl ( String ) — ACL to be used, default is "public-read".
  • contentType ( String ) — Content type of the file.

# (void) delete(bucket, keys, callback)

Delete object(s) from S3. Keys can be a string or an array of strings.
