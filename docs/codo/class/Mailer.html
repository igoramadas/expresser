Overview

EXPRESSER MAILER

Sends and manages emails, with template and tags replacement support. When parsing templates, the tags should be wrapped with normal brackets {}. Example: {contents} The base message template (which is loaded with every single sent message) must be saved as base.html, under the /emailtemplates folder (or whatever folder / base file name you have set on the settings).

# (void) send(options, callback)

Sends an email to the specified address. A callback can be specified, having (err, result).

Parameters:

  • options ( String ) — The email message options
  • callback ( Method ) — Callback (err, result) when message is sent or fails.

Options Hash: (options):

  • body ( String ) — The email body in text or HTML.
  • subject ( String ) — The email subject.
  • to ( String ) — The "to" address.
  • from ( String ) — The "from" address, optional, if blank use default from settings.
  • template ( String ) — The template file to be loaded, optional.

# (Object) createSmtp(options)

Helper to create a SMTP object.

Parameters:

  • options ( Object ) — Options like service, host and port, username, password etc.

Returns:

  • ( Object ) — A Nodemailer SMTP transport object, or null if a problem was found.

# (void) setSmtp(options, secondary)

Use the specified options and create a new SMTP server.

Parameters:

  • options ( Object ) — Options to be passed to SMTP creator.
  • secondary ( Boolean ) — If false set as the main SMTP server, if true set as secondary.
